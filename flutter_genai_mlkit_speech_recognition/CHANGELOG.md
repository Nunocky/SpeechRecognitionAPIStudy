## 0.0.1

* Initial release
* Microphone-based real-time speech recognition
* File-based audio transcription (supports MP3, AAC, FLAC, OGG, WAV, etc.)
* Multi-language support (15 languages)
* Event-based streaming API for recognition results
* Android 12+ (API level 31+) support
* Automatic audio format conversion to 16kHz mono PCM
* Model download management
